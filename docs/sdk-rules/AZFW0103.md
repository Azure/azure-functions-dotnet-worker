# AZFW0103: Invalid Version for Extension Package

| Item | Value |
| - | - |
| **Name** | InvalidExtensionPackageVersion |
| **Category** | Sdk |
| **Severity** | Error |

## Cause

An extension package was resolved with an invalid version.

## Rule Description

Extension packages are resolved by scanning worker packages for declared extensions. These declared extensions must be valid package references.

## How to Fix

Reach out to the author of the worker package which requested the invalid extension package to fix the issue.
