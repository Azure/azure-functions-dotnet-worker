# AZFW0104: Functions Version has Reached End of Life

| Item | Value |
| - | - |
| **Name** | EndOfLifeFunctionsVersion |
| **Category** | Sdk |
| **Severity** | Warning |

## Cause

The property `AzureFunctionsVersion` with value '[VERSION]' is out of support.

## Rule Description

See https://aka.ms/azure-functions-retired-versions for retired function versions.

## How to Fix

Update `AzureFunctionsVersion` property to an in-support value.
