# AZFW0106: Unknown Functions Version Used

| Item | Value |
| - | - |
| **Name** | UnknownFunctionsVersion |
| **Category** | Sdk |
| **Severity** | Error |

## Cause

The property `AzureFunctionsVersion` with value '[VERSION]' is unknown or out of support.

## Rule Description

See https://aka.ms/azure-functions-retired-versions for retired function versions.

## How to Fix

Update `AzureFunctionsVersion` property to an in-support value.
