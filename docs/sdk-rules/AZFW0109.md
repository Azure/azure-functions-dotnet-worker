# AZFW0109: Extension package target framework missing

| Item | Value |
| - | - |
| **Rule ID** | AZFW0109 |
| **Category** | Sdk |
| **Severity** | Error |

## Cause

This SDK error occurs when Azure Functions extension packages are detected without an associated target framework. This means the project is corrupt.

## Rule Description

This is a validation failure for generating the extension project.

## How to Fix

Verify the target `CollectExtensionPackages` has correctly run, and all produced items include a `TargetFramework` metadata.
